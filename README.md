# Homelab Security Journey

## Overview
This repository documents my hands-on journey building a personal homelab to develop practical skills in systems, networking, and cybersecurity.

The goal is to move beyond theory and gain real-world experience by working directly with operating systems, virtual machines, and network configurations.

---

## Objectives
- Build and manage a virtualized lab environment  
- Understand how systems communicate across a network  
- Practice troubleshooting system and connectivity issues  
- Gain exposure to basic security concepts and monitoring  

---

## Current Status
Starting initial setup. Hardware has been acquired and environment planning is in progress.

---

## Planned Setup

### Hardware
- Primary PC (host machine)
- Secondary PC (lab/testing system)

### Virtualization
- VirtualBox or VMware Workstation Player

### Operating Systems
- Windows (for client environment)
- Linux (Ubuntu for system and networking practice)

---

## Lab Roadmap

### Phase 1 – Foundation
- Install virtualization software  
- Create Windows and Linux virtual machines  
- Configure basic networking between systems  

### Phase 2 – Networking & Systems
- Assign IP addresses  
- Test connectivity between machines  
- Explore system processes and configurations  

### Phase 3 – Security Fundamentals
- Monitor system activity and logs  
- Learn basic threat detection concepts  
- Identify normal vs abnormal system behavior
